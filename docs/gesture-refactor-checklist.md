# 手势层重构验收清单（基线 · 阶段 0）

> 用途：手势层重构各阶段结束后的回归基线。每阶段必须跑完全部 6 组并全绿。
> 范围：垂直×3（quick/new/score）、水平×3（phone开/drawer关/scrim关）。
> 不含：roster 行拖拽、long-press、press-feedback、pointer-guard。

## 前置准备

1. 启动：`.\start-lan-preview.cmd --lan`，手机访问局域网 IP（或 USB 直连）。
2. 每测完一组，恢复到该组的初始状态再测下一组。
3. 阈值参考（来自 `drag-gesture.js` / `drawer-gestures.js`）：
   - 起拖阈值 8px、关闭阈值 80px（垂直）/ 50px（水平）、坡度 1.5
   - 抽屉关闭位移 `closedPx = -1.2 × drawer.offsetWidth`

## 通用判定（所有手势共用）

| 项 | 通过 | 失败 |
|---|---|---|
| 起拖前位移 < 8px | 不响应、元素不动 | 元素跟手抖动 |
| 拖拽中 | `transition: none`，元素随手指 | 卡顿/跳变/有动画过渡 |
| 方向相反 | 不位移（被 clamp 锁住） | 反向跟手 |
| 松手未达阈值 | `transition` 还原，元素回弹归位 | 停在原地/直接关闭 |
| 松手达阈值 | 触发对应 open/close | 不触发/误触 |
| 拖拽中页面 | 不滚动、不长按 | 页面滚动/触发长按 |

## V1 · quickPanel 上滑关闭

- **入口**：打开快速面板（quickPanel 可见）。
- **初始状态**：面板可见、`closeDirection = -1`（仅上滑有效）。
- **步骤**：
  1. 在面板上向下滑 < 8px 抬手 → 面板不动、不关闭。
  2. 在面板上向上拖 ≥ 80px 抬手。
- **通过信号**：拖动中面板随手指上移；松手后面板消失（`closeAllCenterPanels` 触发）。
- **失败信号**：下滑也跟手 / < 80px 即关 / 松手不归位 / 拖动时整页滚动。

## V2 · newAssignmentPanel 上滑关闭

- **入口**：打开「新建作业」面板。
- **步骤**：同 V1。
- **通过信号**：同 V1（`closeAllCenterPanels` 触发）。
- **失败信号**：同 V1。

## V3 · scoreSheet 下滑关闭

- **入口**：打开某学生打分面板。
- **初始状态**：`closeDirection = +1`（仅下滑有效）。
- **步骤**：
  1. 在面板上向上滑 < 8px 抬手 → 不动、不关闭。
  2. 在面板上向下拖 ≥ 80px 抬手。
- **通过信号**：拖动中面板随手指下移；松手后面板消失（`closeScoreSheet` 触发）。
- **失败信号**：上滑也跟手 / < 80px 即关 / 松手不归位。

## H1 · phone 右滑开抽屉

- **初始状态**：抽屉关闭、scrim 不可见。
- **排除区**（touchstart 命中以下则不起拖）：`.drawer / .score-sheet / .center-panel / .nav-button / .icon-button / .title-wrap`。
- **步骤**：
  1. 在排除区（如导航按钮）右滑 → 抽屉不动、不误触。
  2. 在空白卡片区域向左滑 < 8px 抬手 → 不动。
  3. 在空白卡片区域向右拖 ≥ 50px 抬手。
- **通过信号**：拖动中抽屉随手指从 `closedPx` 右移到 0；松手后抽屉展开（`openDrawer`）、scrim 淡入；松手后下一张卡片点击被抑制（`setSuppressNextCardClick`）。
- **失败信号**：拖动中触发卡片长按 / 排除区误触 / < 50px 即开 / 松手不归位 / 抽屉已开时仍响应。

## H2 · drawer 自身左滑关抽屉

- **初始状态**：抽屉展开（`drawer.is-open`）。
- **步骤**：
  1. 在抽屉内向右滑 < 8px 抬手 → 不动。
  2. 在抽屉内向左拖 ≥ 50px 抬手。
  3. 向左拖 < 50px 抬手。
- **通过信号**：拖动中抽屉随手指从 0 左移到 `closedPx`；达阈值松手 → 抽屉关闭、scrim 淡出（`closeDrawer`）；未达阈值 → 抽屉回弹展开。
- **失败信号**：右滑也跟手 / < 50px 即关 / 松手不归位 / 关闭后 scrim 残留。

## H3 · scrim 左滑关抽屉

- **初始状态**：抽屉展开、scrim 可见。
- **前置条件**：仅当 `drawer.is-open` 时 scrim 起拖。
- **步骤**：
  1. 抽屉关闭时在 scrim 区域左滑 → 不响应（scrim 不可点）。
  2. 抽屉展开时在 scrim 上向右滑 < 8px 抬手 → 不动。
  3. 在 scrim 上向左拖 ≥ 50px 抬手。
- **通过信号**：拖动中抽屉随手指左移（与 H2 同步位移）；达阈值松手 → 抽屉关闭、scrim 淡出（`closeDrawer`）。
- **失败信号**：抽屉关时 scrim 仍可拖 / 右滑跟手 / < 50px 即关 / 与 H2 位移不同步。

## 跨手势回归（每阶段必跑）

| 项 | 通过 | 失败 |
|---|---|---|
| transition 过渡期（320ms）内再次起拖 | 被 `overlayTransitionBusy` 拦截 | 期间可起拖，出现动画叠加 |
| 任一手势结束后 | 其他 5 组行为不变 | 迁移后串扰 |
| `cachedClosedPx` 串扰（多指快速切换） | 无位移跳变 | 跨实例残留旧值 |
